    Class CRList<T>

    A convergent replicated list.

    Numeric property access reads and mutates the live list projection: list[0] reads the live value reference, list[0] = value writes an entry, and delete list[0] removes one entry. Iteration, find(), and forEach() expose the same live value references. Mutating returned objects directly can mutate replica state without producing a CRDT delta, so callers must isolate values before out-of-band mutation. Local mutations emit delta and change events; remote merges emit change events.
