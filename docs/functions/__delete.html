      Deletes a range from the replica live view.

      With no indexes, the full list is deleted. With only startIndex, all entries from startIndex onward are deleted. With both indexes, the deleted range is [startIndex, endIndex).

      Type Parameters

      • T

      Parameters

      • crListReplica: CRListState<T>

        Replica to mutate.

      • OptionalstartIndex: number

        Inclusive start index. Defaults to 0.

      • OptionalendIndex: number

        Exclusive end index. Defaults to the current list size.

      Returns false | { change: CRListChange<T>; delta: CRListDelta<T> }

      • A local change and gossip delta, or false if nothing was deleted.

      Time complexity: O(d + qk + r), worst case O(n^2)

      • d = distance from cursor to target index
      • q = amount of deleted nodes
      • r = amount of nodes after the deleted range whose indexes must be shifted
      • k = sibling bucket size when deleted entries are removed from buckets

      Space complexity: O(q)
