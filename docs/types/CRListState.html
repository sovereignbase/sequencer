    Mutable CRList replica state.

    blocksById indexes blocks by every contained item id. blocksByPreviousBlockId indexes stable ordering buckets for deterministic flattening. deletedRanges records deleted item ids as sorted id ranges until they are garbage collected through acknowledgement frontiers.
